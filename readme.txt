Mix Master is the wordpress plugin that you need to display mixcloud real-time streaming services.


== Description ==

Mix Master is easy to use and interactive. It quickly allows you to integrate live mixcloud radio shows, Dj mixes and podcasts into your Wordpress.

These are the main options we packed into this extension. Mix Basic Player widget for easy deployment and fast loading times. Mix User Profile widget to easily gather followers and listeners.

Includes TechGasp BANG! Artwork, new player setting for users with beautiful cover artworks. Awesome.

Check the screenshots.

Both the widget and the shortcode framework are fully mobile responsive, a must for any professional website.

== Installation ==

1. Use the Wordpress Installer, Updater
2. Activate in the "Plugins" admin panel.
3. Visit the "Widget Page | Mix Master Widgets" to configure widget settings and your template widget positions.
4. [Mix Master Documentation](https://wordpress.techgasp.com/mix-master-documentation)
